# py-practice

A Python practice repository for refreshing computer science fundamentals and improving practical software engineering skills.

I recently completed an MSc in Artificial Intelligence, which was primarily research-focused. This repository is aimed at strengthening production-style Python skills, debugging habits, and practical tooling for software engineering and data-focused roles.

## Contents

* SQL refresher examples using SQLite
* Python/C speed-up demos using `ctypes`
* Logging examples
* Profiling examples
* Async/`asyncio` examples
* General Python, Git, Linux, and systems-practice notes

## Aim

The main goal is to move from research-style code toward cleaner, more maintainable software engineering practice.

Topics include:

* readable Python scripts
* structured logging
* profiling and performance awareness
* SQL basics
* Python/C interoperability
* asynchronous programming patterns
* Git/GitHub workflow
